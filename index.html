ความรู้ภาษาโปรแกรม

JavaScript
ภาษา script บนฝั่ง client ใช้จัดการ DOM และเพิ่มความโต้ตอบ
  • กำเนิดจาก Netscape ในปี 1995
  • ใช้ในการจัดการองค์ประกอบหน้าเว็บ (DOM manipulation)
  • รองรับ event handling และ AJAX
  • สามารถใช้ฝั่ง server ด้วย Node.js
Python
ภาษาโปรแกรมเชิงวัตถุ ใช้ง่าย นิยมด้าน Data Science และ Web Backend
  • ออกแบบโดย Guido van Rossum ในปี 1991
  • มีไลบรารีครอบคลุม เช่น NumPy, Pandas, Flask, Django
  • ใช้เขียนสคริปต์ และระบบอัตโนมัติ
Java
ภาษาแบบ compiled รันบน JVM เขียนครั้งเดียวรันได้ทุกที่
  • พัฒนาโดย Sun Microsystems ในปี 1995
  • ใช้ใน Android, Enterprise Application, Big Data
  • มี garbage collection และระบบแพ็กเกจ
C++
ภาษาที่ขยายจาก C รองรับ OOP ใช้ในระบบและเกม
  • ออกแบบโดย Bjarne Stroustrup ในปี 1985
  • ชั้นต่ำ ใกล้ฮาร์ดแวร์, ประสิทธิภาพสูง
  • ใช้พัฒนาเกม, ซอฟต์แวร์ระบบ, embedded
Ruby
ภาษา dynamic, ง่ายต่อการอ่าน เข้ากับ Rails ได้ดี
  • สร้างโดย Yukihiro Matsumoto ในปี 1995
  • ใช้ในเว็บแอปด้วย Ruby on Rails
  • มี syntax ที่เป็นมิตรกับผู้ใช้
PHP
ภาษา script บน server นิยมทำเว็บไดนามิก
  • สร้างโดย Rasmus Lerdorf ในปี 1994
  • ฝังใน HTML ทำงานฝั่ง server
  • ใช้กับ CMS เช่น WordPress, Drupal, Joomla
